Computer Networking

Computer networking refers to the practice of connecting multiple computers and devices together to share resources and communicate with one another.

Network

A network is a group of interconnected devices, such as computers, servers, and routers, that are connected together to share information and resources.

Distributed System

A distributed system is a network of computers that work together as a single system to accomplish a common goal.

Uses of Computer Networks

  • File sharing
  • Remote access
  • Email
  • Internet connectivity

Network Topologies

Network topologies refer to the physical and logical layout of a network, such as a star, bus, or ring.

Types of Channels

  • Simplex
  • Duplex
  • Half-duplex

Switching Techniques

Switching techniques refer to the methods used to transfer data between devices on a network, such as circuit switching and packet switching.

Repeaters, Bridge, Gateway

Repeaters, bridges, and gateways are network devices that help to connect different networks together.

Token Passing

Token passing is a method of media access control in which a token, or small data packet, is passed around a network to determine who is able to transmit data.

Protocols

Protocols are set of rules and standards that govern how devices on a network communicate with one another. The ISO-OSI model is a commonly used reference model for describing the different layers of a network protocol.

NIC

NIC (Network Interface Card) is a hardware component that connects a computer to a network.

Serial and Parallel Communication

Serial and parallel communication refers to the method of transmitting data, where serial communication transmits one bit at a time and parallel transmits multiple bits at a time.
